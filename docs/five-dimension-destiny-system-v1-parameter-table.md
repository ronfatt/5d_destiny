# 五维算命系统参数表 V1

## 1. 文档定位

本文件是五维算命系统的核心底层参数表，用于统一以下系统：

- AI 解读系统
- 命运卡牌系统
- Web 评分引擎
- 问卷系统
- 课程与案例体系

本文件采用“引擎版参数”标准，优先考虑可计算、可解释、可校准，而不是纯展示叙事。

## 2. 五维变量总表

| 变量 | 含义 | 来源 | 引擎范围 | 展示方式 |
| --- | --- | --- | --- | --- |
| Structure | 先天结构底盘 | 紫微斗数命盘 | 0-100 | 结构分 |
| Timing | 阶段时运窗口 | 大运/流年/流月 | 0.70-1.30 | 时运系数 |
| Energy | 当下能量扰动 | 三张牌阵/状态输入 | -20.0 至 +20.0 | 能量分 |
| Mind | 心念与认知状态 | 频率系统/问卷/AI问答 | 0.82-1.18 | 心念系数 |
| Action | 现实推进强度 | 行为问卷/行为追踪 | 0-100 | 行动分 / 行动等级 |

说明：

- `Timing` 与 `Mind` 采用收敛后的引擎范围，避免乘数失真。
- `Action` 在引擎内使用 `0-100`，在前端可映射为展示等级。
- `Energy` 内部允许保留一位小数，前端展示时可四舍五入为整数。

## 3. 紫微主星基础分

### 3.1 设计原则

主星基础分用于提供宫位结构的主基线，不直接代表最终宫位分。

设计原则：

- 以 `70` 左右为中轴，避免整体分数普遍过高
- 高低差要能拉开层级，但不能让主星一项吃掉全部解释权
- 最终必须与辅星、四化、状态、煞忌共同作用

### 3.3 双主星计算

若宫位存在双主星：

`MainStarBase = (StarA + StarB) / 2`

示例：

- 武曲 `80`
- 贪狼 `69`

则：

`(80 + 69) / 2 = 74.5`

## 6. 宫位原始分公式

### 6.1 公式

`PalaceRawScore = MainStarBase + SupportStarDelta + TransformationDelta + ShaJiDelta + StatusDelta`

### 6.2 状态修正建议

| 状态 | 修正 |
| --- | --- |
| 入庙 | +6 |
| 旺 | +4 |
| 平 | 0 |
| 陷 | -6 |

### 6.3 截断规则

`PalaceScore = min(100, max(0, PalaceRawScore))`

说明：

- 宫位分必须限制在 `0-100`。
- 不允许出现负分宫位或超过 `100` 的宫位。

### 6.4 示例

命宫：

- 主星：武曲 + 贪狼
- 左辅
- 化禄
- 火星

计算：

- 主星：`(80 + 69) / 2 = 74.5`
- 左辅：`+8`
- 化禄：`+10`
- 火星：`-8`

则：

`PalaceRawScore = 74.5 + 8 + 10 - 8 = 84.5`

最终：

`PalaceScore = 84.5`

## 12. 最终命运公式

### 12.1 展示公式

`Destiny Score = ((Structure × Timing) + Energy) × Mind + ActionDisplay`

说明：

- 该公式适合课程与对外表达。
- `ActionDisplay` 为 `-10 至 +10` 的展示值。

### 12.2 引擎公式

为保证量纲一致，v1.0 引擎统一采用标准化形式：

- `S = Structure / 100`
- `T = Timing`
- `E = 1 + (Energy / 100)`
- `M = Mind`
- `A = Action / 100`

中间指标：

- `BasePotential = S × T`
- `StateMomentum = E × M`
- `ExecutionPower = A × M`

最终：

`TrendScore = round((BasePotential × 50) + (StateMomentum × 20) + (ExecutionPower × 30))`

说明：

- 对外保留简单公式，对内执行稳定公式。
- 这样既方便传播，也便于系统实现和校准。

## 15. 当前版本结论

五维系统 v1 的底层卖点不是“神秘”，而是“多源输入下的可解释趋势计算”。

它的核心组合是：

- 紫微斗数：提供结构底盘
- 时运系统：提供阶段窗口
- 卡牌系统：提供能量扰动
- 心念系统：提供认知调节
- 行动系统：提供现实转化

因此，v1 的本质不是传统占卜，而是一个可计算的命运趋势引擎。
