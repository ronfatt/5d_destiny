# AI 报告输出模板 V1

## 1. 文档定位

本文件定义五维命运系统的 AI 报告输出模板。

目标是把平台中的结构化输入稳定转换为：

- 用户可读报告
- 单主题分析
- 多主题综合分析
- 免费版与付费版内容结构

本文件服务于：

- AI 解读引擎
- 报告生成系统
- Web 报告页
- PDF 报告导出
- 会员日报 / 周报 / 月报模板

## 2. 设计目标

AI 报告模板必须满足以下要求：

- 可复用
- 可追溯
- 可控
- 可扩展
- 不胡编

AI 的职责是解释结构化结果，而不是创造未经计算支持的结论。

## 3. 输入来源

报告生成时，AI 必须优先读取以下结构化输入：

- 用户问题
- 主题类型
- 命盘结果
- 卡牌结果
- 五维变量
- 趋势分数
- 风险标记
- 主导变量
- 建议优先级

## 4. 标准输入结构

建议统一输入格式：

```json
{
  "question": "今年事业发展如何？",
  "theme": "career",
  "structure": 81,
  "timing": 1.12,
  "energy": 10.2,
  "mind": 1.06,
  "action": 72,
  "trendScore": 94,
  "trendLevel": "高势期",
  "dominantFactor": "结构主导",
  "riskFlags": ["短期压力偏高"],
  "cards": {
    "archetype": "武曲执行",
    "energy1": "机会",
    "energy2": "压力",
    "event": "合作"
  },
  "breakdown": {
    "basePotential": 0.91,
    "stateMomentum": 1.17,
    "executionPower": 0.76
  },
  "advicePriority": [
    "聚焦关键项目",
    "用协作替代单打独斗",
    "控制短期压力负荷"
  ]
}
```

## 5. 报告层级设计

建议分成四类输出层级：

1. 轻量摘要
2. 标准单主题报告
3. 完整四主题报告
4. 会员周期报告

## 6. 轻量摘要模板

### 6.1 使用场景

- 免费体验
- 日报入口
- 快速抽牌结果
- Bot 返回结果

### 6.2 输出结构

1. 一句话结论
2. 当前状态
3. 一条风险提示
4. 一条行动建议

### 6.3 模板示例

```text
当前事业趋势处于高势期，结构与时运都偏强。
命运卡提示你正站在机会窗口前，但短期压力不小。
当前风险是推进过快导致负荷过重。
建议先聚焦最关键的一项动作，不要同时铺开太多方向。
```

## 7. 标准单主题报告模板

### 7.1 使用场景

- 单次占卜
- 单主题付费解读
- 快速咨询报告

### 7.2 标准结构

1. 主题标题
2. 总体趋势
3. 五维拆解
4. 卡牌解析
5. 风险提醒
6. 行动建议

### 7.3 标题模板

`{ThemeName} 命运趋势报告`

示例：

- 事业命运趋势报告
- 财富命运趋势报告
- 感情命运趋势报告
- 健康命运趋势报告

### 7.4 总体趋势模板

模板：

```text
当前 {theme} 趋势为 {trendLevel}。
从五维结果来看，{dominantFactor} 是当前最主要的驱动因素。
```

扩展写法：

- 若 `Structure` 高：强调底盘强
- 若 `Timing` 高：强调窗口打开
- 若 `Energy` 高：强调状态助推
- 若 `Mind` 低：强调内在状态折损
- 若 `Action` 低：强调潜力未兑现

### 7.5 五维拆解模板

建议固定顺序：

1. Structure
2. Timing
3. Energy
4. Mind
5. Action

模板：

```text
结构面上，{structureSummary}。
时运面上，{timingSummary}。
能量面上，{energySummary}。
心念面上，{mindSummary}。
行动面上，{actionSummary}。
```

示例：

```text
结构面上，你的事业底盘较强，说明并不是没有能力，而是本来就具备一定承接机会的条件。
时运面上，今年属于偏上升阶段，窗口不是最极端的爆发期，但已经明显优于停滞期。
能量面上，当前机会与压力并存，说明前方有入口，但推进成本不低。
心念面上，你整体状态偏稳定，说明当前并不是被情绪完全拖住。
行动面上，你已经具备一定执行度，但还没有达到完全拉满的状态。
```

### 7.6 卡牌解析模板

模板：

```text
这次牌阵中，主星卡 {archetype} 代表事情背后的核心驱动力。
{energy1} 与 {energy2} 描述当前的主要状态与阻力。
事件卡 {event} 则指出现实层面的触发点。
```

扩展说明：

- 主星卡先解释“是什么力量在主导局势”
- 能量卡解释“现在是什么状态”
- 事件卡解释“外部世界发生了什么”

### 7.7 风险提醒模板

模板：

```text
当前最需要注意的是：{risk1}。
另外，{risk2}。
```

若风险只有一项，则只输出一条。

风险文案建议来源：

- `Timing < 0.90`
- `Energy <= -8`
- `Mind <= 0.94`
- `Action <= 40`
- 双极卡出现负向结果

### 7.8 行动建议模板

建议固定为三段：

1. 未来 7 天
2. 未来 30 天
3. 当前最优策略

模板：

```text
未来 7 天：{advice7d}
未来 30 天：{advice30d}
当前最优策略：{bestStrategy}
```

行动建议必须满足：

- 可执行
- 不抽象
- 与当前主题相关
- 与 Action 或 Mind 的短板相关

## 8. 完整四主题报告模板

### 8.1 使用场景

- 深度报告
- 年度命运报告
- 高价付费产品

### 8.2 报告结构

1. 封面
2. 命盘概览
3. 五维总说明
4. 四主题总览
5. 事业分析
6. 财富分析
7. 感情分析
8. 健康分析
9. 风险总表
10. 行动路线图

### 8.3 四主题总览模板

```text
从整体趋势看，你当前最强的领域是 {bestTheme}，最需要修正的领域是 {weakestTheme}。
这意味着你当前的主线不应平均发力，而应采取“主攻一项、修复一项、维持两项”的策略。
```

### 8.4 各主题摘要模板

每个主题固定输出：

- 趋势等级
- 主导变量
- 一句话判断

示例：

```text
事业：高势期，结构主导。适合主动推进并扩大成果。
财富：成长区，行动主导。结果更多取决于执行纪律。
感情：整理期，心念主导。当前重点不是推进，而是修复内在反应模式。
健康：成长区，行动主导。稳定作息将直接提高整体状态。
```

## 10. 文案生成规则

### 10.1 必须做的事

AI 必须：

- 用结构化输入组织结论
- 明确指出主导变量
- 区分长期结构与短期状态
- 区分机会与风险
- 给出行动建议

### 10.2 不允许做的事

AI 不允许：

- 虚构没有出现的主星或牌义
- 把轻微信号夸大成宿命结论
- 只讲情绪安慰，不给动作建议
- 将分数矛盾解释为绝对确定性

### 10.3 风格要求

文案应：

- 清楚
- 稳定
- 克制
- 具体
- 不夸张

文案不应：

- 神神叨叨
- 绝对化
- 恐吓用户
- 写成空泛鸡汤

## 11. 模板变量建议

建议系统预先生成以下中间变量，供 AI 使用：

- `structureSummary`
- `timingSummary`
- `energySummary`
- `mindSummary`
- `actionSummary`
- `dominantFactor`
- `riskFlags`
- `advice7d`
- `advice30d`
- `bestStrategy`
- `bestTheme`
- `weakestTheme`

这样 AI 就不是从零生成，而是在结构基础上完成语言包装。

## 12. 示例单主题报告

### 输入示例

```json
{
  "theme": "career",
  "question": "今年事业发展如何？",
  "structure": 81,
  "timing": 1.12,
  "energy": 10.2,
  "mind": 1.06,
  "action": 72,
  "trendScore": 94,
  "trendLevel": "高势期",
  "dominantFactor": "结构主导",
  "riskFlags": ["短期压力偏高"],
  "cards": {
    "archetype": "武曲执行",
    "energy1": "机会",
    "energy2": "压力",
    "event": "合作"
  },
  "advicePriority": [
    "聚焦关键项目",
    "用协作替代单打独斗",
    "控制短期压力负荷"
  ]
}
```

### 输出示例

```text
事业命运趋势报告

当前事业趋势为高势期。
从五维结果来看，结构主导是当前最主要的驱动因素。

结构面上，你的事业底盘较强，这说明你不是没有机会，而是本来就具备承接机会的能力。
时运面上，今年属于明显上升阶段，机会窗口已经打开。
能量面上，这次牌阵显示机会与压力并存，说明入口存在，但推进成本不低。
心念面上，你当前整体状态偏稳定，说明你仍能保持相对清晰的判断。
行动面上，你已经具备稳定执行力，但还没有完全拉满。

这次牌阵中，主星卡武曲执行代表事情背后的核心驱动力是行动与推进。
机会卡说明现实中确实存在新的入口，压力卡提醒你短期负担不小，合作事件卡则说明你更适合通过协作来放大结果，而不是独自硬扛。

当前最需要注意的是短期压力偏高。如果节奏失控，原本的优势可能被消耗掉。

未来 7 天：聚焦一个最关键的项目，不要同时推进太多分支。
未来 30 天：建立更稳定的协作节奏，让资源和执行形成闭环。
当前最优策略：借势推进，但不要用透支的方式换结果。
```

## 14. 当前版本结论

AI 报告模板的作用不是代替模型，而是把模型结果稳定地转译成用户能理解的语言。

只有当模板结构稳定后，AI 解读系统才不会一会儿像报告，一会儿像闲聊，一会儿像玄学文案。
